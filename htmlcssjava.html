Hyper Text Markup Language

HTML, or HyperText Markup Language, is the standard markup language used to create and design web pages. It provides a structure for web content by using a system of elements or tags that define the various components on a webpage. HTML elements can include text, images, links, forms, and other types of multimedia.

Cascading Style Sheets

CSS or Cascading Style Sheets is a stylesheet language used to add styles to the HTML document. It describes how HTML elements should be displayed on the web page.

Why learn CSS?

  1. Control all design-related aspects of your website, including typography, colours, page layouts and any other visual aspects of your website.
  2. Control and adjust the design and formatting of your entire website (no matter how big or small it is) from a single spot.
  3. Create the visual design of web content.
  4. Adapt web pages to different devices and browsers.
  5. Improve the user experience of websites, apps, and more.

JavaScript

JavaScript is the most popular scripting language for the Web. It is easy to learn, lightweight, cross-platform, single-threaded, and interpreted compiled language. It is widely used for web development, both on the client side and server side. As a scripting language, it plays an important role in web browsers, allowing real-time modification of web page content and enhancing user experience.

Why learn JavaScript?

  1. JavaScript enables dynamic and interactive web experiences, allowing users to engage with content in real-time.
  2. As a versatile scripting language, JavaScript empowers developers to build various applications, from web development to mobile and desktop apps.
  3. With JavaScript, you can execute code directly in the browser, reducing server load and enhancing user experience.
  4. JavaScript boasts a vast community of developers and resources, facilitating learning and problem-solving.
  5. Proficiency in JavaScript opens doors to diverse career opportunities, including web development, game development, and even server-side programming with frameworks like Node.js.
